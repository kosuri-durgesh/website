# Delicious Bites Restaurant Website

A modern, responsive website for Delicious Bites restaurant featuring an elegant design and comprehensive information about our services.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling navigation with active section highlighting
- **Mobile Menu**: Hamburger menu toggle for mobile devices
- **Menu Filtering**: Filter menu items by category (All, Appetizers, Main Courses, Desserts)
- **Scroll Animations**: Smooth fade-in animations as sections come into view
- **Hero Section**: Eye-catching banner with call-to-action
- **About Section**: Information about the restaurant and its features
- **Menu Section**: Categorized menu with appetizers, main courses, and desserts
- **Contact Section**: Restaurant address, phone number, and hours
- **Modern Styling**: Clean, professional design with attractive color scheme

## Technologies Used

- HTML5
- CSS3 with Flexbox and animations
- Vanilla JavaScript (ES6+)
- Responsive design principles

## Files

- `index.html` - Main HTML structure
- `style.css` - CSS styling and animations
- `script.js` - JavaScript functionality
- `README.md` - This documentation

## How to View

Simply open `index.html` in your web browser to view the website.

## JavaScript Features

- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Mobile Menu Toggle**: Hamburger menu for mobile navigation
- **Menu Filtering**: Interactive buttons to filter menu categories
- **Scroll Animations**: Elements animate in as they enter the viewport
- **Active Navigation**: Current section is highlighted in the navigation
- **Form Validation**: Ready for contact form validation (expandable)

## Customization

- Edit `index.html` to change content
- Modify `style.css` to update styling and animations
- Update `script.js` to add new functionality
- Replace the background image URL in the hero section for a custom image

## Sections

1. **Home**: Welcome message and hero banner
2. **About**: Restaurant description and key features
3. **Menu**: Interactive menu with filtering options
4. **Contact**: Location, phone, and operating hours

Enjoy exploring Delicious Bites!
